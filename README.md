## ✅ Task 1: Exploring and Visualizing a Dataset (Iris Dataset)

### 🎯 Objective

To understand how to load, explore, and visualize a dataset using Python.

### ⚙️ Approach

* Loaded dataset using pandas
* Explored structure using `.shape`, `.columns`, `.head()`
* Created visualizations:

  * Scatter Plot
  * Histogram
  * Box Plot

### 📈 Results & Insights

* Identified relationships between features
* Observed clear separation between species
* Understood data distribution and outliers

## ✅ Task 2: Credit Risk Prediction (Loan Dataset)

### 🎯 Objective

To predict whether a loan applicant is likely to default.

### ⚙️ Approach

* Checked and confirmed no missing values
* Performed Exploratory Data Analysis (EDA)
* Converted categorical data using encoding
* Trained Logistic Regression and Decision Tree models
* Evaluated using accuracy and confusion matrix

### 📈 Results & Insights

* Credit score and income significantly affect loan default
* Model achieved good accuracy
* Confusion matrix helped evaluate prediction performance

---

## ✅ Task 3: Customer Churn Prediction (Bank Dataset)

### 🎯 Objective

To identify customers likely to leave the bank.

### ⚙️ Approach

* Cleaned dataset and removed unnecessary columns
* Applied Label Encoding and One-Hot Encoding
* Trained Random Forest model
* Evaluated model performance
* Analyzed feature importance

### 📈 Results & Insights

* Age, balance, and geography are key factors influencing churn
* Model successfully identifies at-risk customers
* Feature importance helps understand customer behavior
